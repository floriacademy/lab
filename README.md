# FLORIA Academy Lab

**L'eclosion de votre reussite**

Bienvenue dans le laboratoire FLORIA Academy. Cet espace est dedie aux talents IT du Maroc et d'Afrique francophone pour realiser, soumettre et faire valider leurs projets par notre mentor IA.

## Comment ca marche

1. **Forkez** ce repo sur votre compte GitHub
2. **Creez** votre projet dans `talents/<votre-username>/<nom-projet>/`
3. **Developpez** en suivant le template dans `templates/starter-project/`
4. **Soumettez** une Pull Request vers ce repo
5. **L'IA mentor** evalue automatiquement votre code et poste un feedback
6. **FLORIA** valide et votre projet rejoint le showcase

## Structure

```
talents/
  <username>/
    <project-name>/
      README.md        # Description du projet
      package.json     # Dependencies
      src/             # Code source
      tests/           # Tests
```

## Criteres d'evaluation

Notre IA mentor evalue sur 100 points :

| Critere | Points | Description |
|---------|--------|-------------|
| Qualite du code | 25 | Nommage, structure, DRY, SOLID |
| Documentation | 20 | README complet, commentaires, JSDoc |
| Tests | 20 | Tests presents, passent, couverture |
| Securite | 15 | Pas de secrets, deps a jour, OWASP |
| Fonctionnalite | 20 | Le projet fait ce qu'il promet |

Score minimum pour validation : **60/100**

## Domaines acceptes

Cloud, Infrastructure, Developpement Web, Mobile, Data, IA/ML, Design UX/UI, Architecture, Gestion de Projet, Securite, DevOps.

## Les 16 Principes FLORIA

Chaque projet doit etre : **Securise, Fiable, Honnete, Utile, Objectif, Coherent, Tres simple, Intuitif, Precis, Complet, Performant, Autonome, Parametrable, Evolutif, Accessible et Predictif.**
